
# LemongProject Front-End
LemongProject는 일정관리 및 커뮤니티 서비스 입니다. 
### 접속주소

  [레몽프로젝트](http://www.lemongproject.site)
  
### 기술 스택

  React
  Redux
  Axios

### 설치 및 사용 방법
``` bash
  Copy code
  git clone https://github.com/woongwhee/lemongproject-front.git
  cd lemongproject-front
  npm install
  npm start
```
